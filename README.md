## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

--> NOTED!! Jika command diatas tidak bisa dijalankan maka jalankan command dibawah ini <--

```bash
npm i
# or
yarn i
# or
pnpm i
# or
bun i
```

--> Jika error *Error: Failed to fetch data!* muncul maka buka file .env.example <--
